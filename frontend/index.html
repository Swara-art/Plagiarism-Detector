How It Works

Three detection modes. One explainable report.

🔍

Semantic Text Analysis

Sentence-Transformers encode meaning, not just words. Paraphrasing is caught even without shared vocabulary.

🌳

AST Code Comparison

Tree-sitter parses code into Abstract Syntax Trees. Renamed variables and reordered logic won't hide plagiarism.

✍️

Handwriting OCR

Tesseract OCR converts scanned handwritten pages into digital text, then runs the full similarity pipeline.

🧠

Explainable Reports

Every flagged section tells you exactly why it was flagged — not just a score, but a clear human-readable reason.

📚

Citation Suggestions

When sources are matched, APA-style citation formats are automatically generated for proper attribution.

⚡

Vector Search

ChromaDB stores semantic embeddings for lightning-fast cosine similarity search across thousands of documents.
